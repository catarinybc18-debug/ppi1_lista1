Diferença web e internet

A Internet é a infraestrutura global de redes que conecta computadores no mundo todo. Já a Web (World Wide Web) é um serviço que funciona dentro da Internet. A Web usa navegadores para acessar sites e páginas por meio de links. Ou seja, a Internet é a “rede física”, e a Web é uma forma de usar essa rede.

🌐 História da Internet e Web

A Internetsurgiu como um projeto militar, enquanto a Web foi criada depois para facilitar o acesso às informações.

📜 Linha do Tempo

  1. 1969 - Criação da ARPANET
  2. 1983 - Surgimento do protocolo TCP/IP
  3. 1991 - Criação da Web
  4. 2000+ - Redes sociais
